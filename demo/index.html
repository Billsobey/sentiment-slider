Sentiment Slider Demo

An interactive, touch-friendly sentiment capture component

Demo Controls

Key Features

  • Dynamic background colors reflect sentiment intensity
  • Ripple animation on slider release
  • Real-time feedback text updates
  • Mobile-optimized touch interactions
  • "Next question" button appears after interaction

Implementation Example


import { SentimentSlider } from '@sentiment-slider/react';

function FeedbackForm() {
  const handleConfirm = (value) => {
    console.log(`User sentiment value: ${value}`);
    // Handle confirmation...
  };

  return (
    <SentimentSlider 
      onConfirm={handleConfirm}
      initialValue={50}
    />
  );
}
